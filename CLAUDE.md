# kinotos — サイト設計ガイド

## コンセプト

**「感性を、定量化する。」**

Data Scientist である kinotos が、主観的な体験（サウナ・音楽・コーヒー・クラフトビール）を客観的に記述・分析・共有するための個人サイト。
感性と数字の緊張関係そのものが、このサイトのアイデンティティ。

---

## デザイン世界観

### ビジュアルテイスト

**淡くて眩しい、花が散るような世界観。**

- **パステルの水彩感** — くすんだ桜色・ダスティブルー・ミントグリーン・ラベンダーが混在。高彩度ではなく、霞んだ淡さ
- **花びらが舞い散る動き** — 常時、画面上から花びら型のシェイプがゆっくり落ちる。風景の中にいる感覚
- **水彩ブロブ** — 背景に大きくぼかしたカラーブロブが漂い、ページに奥行きをつける
- **有機的 × 構造的** — 生き物的な曲線（ブロブ、花びら）と、データ的なグリッド・ラインが共存する
- **静謐でありながら色が豊か** — 動きはゆっくり、色は散りばめる。うるさくなく、でも眩しい

### 参照イメージ
- 日常のフィルム写真グリッド（温かみ、手触り感）

### タイポグラフィ
- 見出し・名前：**EB Garamond italic**（詩的・クラシック・軽やか）
- 本文・ラベル：**Inter**（構造的・読みやすい）
- 見出し：letter-spacing マイナス、line-height 0.88〜1.0
- ラベル：10〜11px、letter-spacing 0.2〜0.35em、text-transform uppercase
- 本文：12〜13px、color: var(--muted)、line-height 1.9〜2.0

### ヒーロー名のグラデーション
```css
background: linear-gradient(135deg, #c0a8d8 0%, #e8b4c0 25%, #aec8d8 50%, #c8dca8 75%, #f0c4a0 100%);
background-size: 300% 300%;
animation: gradshift 8s ease infinite;
/* gradshift: background-position 0%→100%→0% */
```

### アニメーション・インタラクション
- **花びら落下** — JS生成の28個のシェイプ、10色からランダム、10〜28秒でゆっくり落下・回転
- **水彩ブロブ** — 4つ、blur(80px)、12秒でゆらゆら漂う（blobdrift）
- **カスタムカーソル** — 6pxドット + lerp 0.08の遅延リング（28px→ホバー時56px）
- **スクロール reveal** — IntersectionObserver、兄弟要素を120ms間隔でstagger、cubic-bezier(.16,1,.3,1) 1s
- **パラレックス** — hero名がスクロール量×0.15でゆっくり上昇
- **easing** — 必ず cubic-bezier(.16,1,.3,1)（spring感）か ease。linear は花びら落下のみ

### やってはいけないこと
- 高彩度な色（パステルの霞み感を壊す）
- アニメーションが速すぎる・派手すぎる（全体的にゆっくり）
- 情報を詰め込みすぎる（余白は意味を持つ）
- コーポレートっぽい表現・レイアウト
- 花びら・ブロブを消す（これがサイトの呼吸）
